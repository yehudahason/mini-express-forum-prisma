### Mini Forum with Express js Postgres db and Prisma ORM

- Create .env file like env.example

- install 

```
 npm install
```

- Set up database with  the prisma shcema

- run dev Server

```
npm run dev
```

- Open http://localhost:4444/

- For production 

```
 npm run build
```

- For prisma studio

```
npx prisma studio 
```

- In DB forum table add rows for your own forums
